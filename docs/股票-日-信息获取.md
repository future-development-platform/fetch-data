# 股票日信息获取说明

### 参数说明

|字段|说明|
|----|----|
|时间(date) | 当前时间|
|股票代码(code) | `sz.股票代码`、`sh.股票代码`|
|开盘价(open)||
|最高价(high)||
|最低价(low)||
|收盘价(close)||
|前收盘价(preclose)||
|成交量(volume) | 累计单位 - 股|
|成交额(amount) | 单位  -  人民币元|
|复权状态(adjustflag) | 1 - 后复权、2 - 前复权、3 - 不复权|
|换手率(turn) | (指定交易日的成交量(股) / 指定交易日的股票流通股总股数(股)) * 100%|
|交易状态(tradestatus) | 1 - 正常交易、0 - 停牌|
|涨跌幅(pctChg) | 涨跌幅(百分比)  -  日涨跌幅 = [(指定交易日的收盘价 - 指定交易日前收盘价) / 指定交易日收盘价] * 100%|
|滚动市盈率(peTTM) | (指定交易日的股票收盘价 / 指定交易日的每股盈余TTM) = (指定交易日的股票收盘价 * 截至当日公司总股本) / 归属母公司股东净利润TTM |
|市净率(pbMRQ) | (指定交易日的股票收盘价 / 指定交易日的每股净资产) = 总市值 / (最近披露的归属母公司股东的权益 - 其他权益工具) |
|滚动市销率(psTTM) | (指定交易日的股票收盘价 / 指定交易日的每股销售额) = (指定交易日的股票收盘价 * 截至当日公司总股本) / 营业总收入TTM |
|滚动市现率(pcfNcfTTM) | (指定交易日的股票收盘价 / 指定交易日的每股现金流TTM ) = (指定交易日的股票收盘价 * 截至当日公司总股本) / 现金以及现金等价物净增加额TTM |
|是否是ST股(isST) | 1 - 是, 0 - 否|
